文件资源库

访问文件地址: https://cdn.jsdelivr.net/gh/lin09/images/${hashPath}

hash = 32位的文件MD5hash

hashPath = hash.replace(/^(.{2})(.{2})/, "$1/$2/")

例如：

hash -> e5439d2a29828e6cd27695149658fdca

hashPath -> e5/43/9d2a29828e6cd27695149658fdca

文件地址 -> https://cdn.jsdelivr.net/gh/lin09/images/e5/43/9d2a29828e6cd27695149658fdca
